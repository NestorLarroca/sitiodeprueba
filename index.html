HTML

HTML (HyperText Markup Languaje) es un lenguaje de etiquetas utilizado para crear páginas web.

Sirve como el lenguaje fundamental para estructurar y organizar el contenido web.

HTML es esencial para crear la estructura básica de una página web.

HTML proporciona la base para e diseño y funcionalidad de una página web.

CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo.

Se utiliza para controlar la presentación y diseño de páginas web.

CSS sirve para aplicar estilos visuales a elementos HTML.

CSS se utiliza para mejorar la apariencia y el diseño de una página web.

JS

JS (JavaScript) es un lenguaje de programación.

Permite implementar funciones complejas en páginas web.

Se utiliza para hacer que las páginas web sean dinámicas.

Los usuarios pueden así interactuar con una página gracias a este lenguaje.
